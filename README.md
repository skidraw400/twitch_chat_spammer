# twitch_chat_spammer

Just download [html file](https://raw.githubusercontent.com/skidraw400/twitch_chat_spammer/main/twitch_chat_spammer.html) and run it on your pc. Using [tmi.js](https://github.com/tmijs/tmi.js/) library.
